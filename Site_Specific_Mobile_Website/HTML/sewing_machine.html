Sewing Machine Guide

Learn more about our sewing machine:

Brother Sewing Machine

We have four standard sewing machines in the makerspace for student use. These machines are used to sew thin fabrics such as quilt weight cotton, felt, linen, and polyester. The Brother CS7000X to an external site. has 70 built in stitches and a variety of accessory sewing feet.

When not in use, the machines are stored on the shelf near the sewing area. When using the sewing machines, you'll want to also check out a sewing kit that contains all the necessary tools and supplies.

Sailrite Fabricator Industrial Sewing Machine

The Sailrite is a full-size industrial, straight stitch sewing machine with a compound walking foot that is powered by a 550 watt, 3/4 horsepower motor. This machine is designed to sew heavy weight materials such as canvas, upholstery, denim, and lightweight leather.
