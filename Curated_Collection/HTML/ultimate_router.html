Installation Steps:

Proxmox Installation

Installing Proxmox into Intel NUC:

1. Ensure your Intel NUC is powered on and connected to the internet.

2. Download the Proxmox ISO from the official website: https://www.proxmox.com/proxmox-ve.

3. Create a bootable USB drive with the Proxmox ISO using a tool like Rufus or Etcher.

4. Insert the USB drive into the Intel NUC and boot from it.

5. Follow the on-screen instructions to install Proxmox on the Intel NUC.

OPNSense Installation

To install OPNSense on Proxmox virtual environment:

1. Ensure Proxmox is installed and running on your Intel NUC.

2. Download the OPNSense ISO from the official website: https://opnsense.org/.

3. Create a new virtual machine in Proxmox and mount the OPNSense ISO.

4. Install OPNSense following the on-screen instructions.

Pihole Installation

To install Pihole on Proxmox virtual environment:

1. Ensure Proxmox is installed and running on your Intel NUC.

2. Create a new virtual machine in Proxmox and mount the Pihole ISO.

3. Install Pihole following the on-screen instructions.
