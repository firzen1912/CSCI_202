Project List

Helix V1 Keyboard

This is a Do-It-Yourself (DIY) split ortho-linear keyboard designed with the aim of reducing the constraints imposed by a standard keyboard setup. The Helix V1 Keyboard offers a unique and customizable experience for users seeking improved ergonomics and efficiency in their typing experience. By splitting the keyboard into distinct halves, each accommodating a set of keys, this design promotes a more natural hand positioning, potentially minimizing strain and discomfort associated with traditional keyboards. The ortho-linear layout arranges keys in a grid, providing a symmetrical and logical configuration that some users find enhances typing speed and accuracy. Whether you're a keyboard enthusiast or someone looking for a more ergonomic typing solution, the Helix V1 Keyboard represents a creative and functional approach to redefine the conventional keyboard experience.

Helix V2 Keyboard

Embarking on the next level of keyboard evolution, the Helix V2 Keyboard introduces several cutting-edge features. With Bluetooth wireless capability, it liberates your workspace, enabling cable-free connections. Manage multiple devices simultaneously, streamlining your workflow with enhanced efficiency. The integrated battery provides unparalleled portability, allowing you to work on the go without constraints. Additionally, the keyboard boasts hot-swappable processors and keyswitches, empowering you to customize your typing experience effortlessly. These advancements collectively redefine the keyboard landscape, offering a seamless blend of innovation and user-centric design.

Ultimate Router

Embark on a journey of network security with the Ultimate Router. This extraordinary setup leverages the power of an Intel NUC NUC6i7KYK within a Proxmox virtual environment, seamlessly integrating essential components to meet all your security needs in one robust router machine. The virtual environment hosts OPNSense for advanced routing capabilities, Pihole for efficient ad-blocking, and NordVPN for secure and private connections. This cohesive setup ensures that your network is fortified against potential threats, providing a comprehensive and efficient solution for all your security and routing requirements. Follow the steps of this groundbreaking project to create a centralized and powerful network hub that brings together intelligence, versatility, and security under one roof.

Mini Kali

Embark on the world of cybersecurity with Mini Kali, a portable Kali Linux machine ingeniously crafted on a Raspberry Pi 3B+. This project goes beyond conventional setups by incorporating its own dedicated mini UPS unit, allowing the Mini Kali to charge and be used simultaneously. The Raspberry Pi serves as a compact yet powerful platform for running Kali Linux, a renowned penetration testing and ethical hacking distribution. With the added versatility of portability and the resilience provided by the mini UPS, Mini Kali becomes a formidable tool for cybersecurity professionals on the go. Follow the step-by-step guide of this project to build your own Mini Kali, empowering you with a portable and self-sustaining cybersecurity solution.
